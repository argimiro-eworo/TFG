INTRODUCCIÓN
¶

El objetivo de este libro electrónico es reunir una serie de ejemplos ilustrativos que demuestren las distintas posibilidades que ofrece la tecnología jupyter y jupyter books para publicación de contenidos de proyectos de ciencias de datos y/o machine learning.

Dichos ejemplos están basados en la reproducibilidad científica. La reproducibilidad es la característica de un proyecto de poder ser recreado, siguiendo los mismos pasos detallados y con los mismos datos y/o distintos, llegando siempre a resultados idénticos.

Para eso vamos a ilustrar los siguientes proyectos:

  • End To End Machine Learning: Analizaremos, con datos proporcionados, varios algoritmos de predicción para los precios de viviendas en la costa de california.

  • El Tiempo En Nueva York: Con datos brutos del tiempo en Nueva York y haciendo uso de la herramienta Altair, crearemos una gráfico interactivo que relaciona los meses del año y las temperaturas.

  • Citi Bike en Nueva York: Con datos brutos del sistema de intercambio de bicicletas en NY y haciendo uso de la herramienta Geopandas, ilustraremos sobre un mapa interactivo, la localización de cada estación así como información del ńumero de llegadas y salidas de bicis en cada una.
